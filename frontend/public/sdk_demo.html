Core API Endpoints

POST /api/explain

Generate adaptive explanations based on student profile and learning history.

const response = await fetch('/api/explain', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ question: 'Explain matrix multiplication', student_id: 'student_123', grade_level: '10' }) });
POST /api/generate-video

Generate professional educational videos with Manim integration.

import requests response = requests.post('https://api.snaplearn.ai/generate-video', { 'topic': 'factorials', 'student_id': 'student_123', 'quality': 'high', 'include_captions': True })
POST /api/quiz/generate

Create adaptive quizzes tailored to student weaknesses and learning style.

curl -X POST https://api.snaplearn.ai/quiz/generate \ -H "Content-Type: application/json" \ -d '{ "topic": "matrices", "num_questions": 5, "difficulty": "adaptive", "student_id": "student_123" }'

Integration Examples

React Integration

import { SnapLearnSDK } from '@snaplearn/react-sdk'; function TutorComponent() { const { explainTopic, loading } = SnapLearnSDK({ apiKey: process.env.SNAPLEARN_API_KEY, studentId: 'student_123' }); return ( <TutorInterface onQuestion={explainTopic} loading={loading} adaptive={true} /> ); }

Python Integration
